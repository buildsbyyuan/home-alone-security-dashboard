## The Emotional Role: User Advocate  

We explored what Kevin feels when he is home alone.

**Key insights**  
- Fear of burglars  
- Feeling forgotten  
- Wanting independence  
- Needing reassurance  
- Wanting to feel clever and brave  

**This shaped**  
- Friendly, childlike language  
- Humorous tone that reduces fear  
- Bright visuals  
- Simple, confidence boosting interactions

- A text input for AI features  
- Fully working interactions
